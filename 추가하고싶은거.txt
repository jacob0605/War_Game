

나중에 ppt에 있는 기능들을 다 구현하면 추가하고 싶은 것들 작성

1. 화면 출력 방식 변경
    C 프로그래밍 수업 과제에서 받은 코드를 활용, 화면 출력 방식 변경
    지금은 밑에서부터 계속 스크롤 되는 형식임.
    이걸 고정된 하나의 화면으로 변경
    C프 과제처럼.

2. 복수의 게임 파일
    게임 진행 상황을 하나의 파일에 저장하는 것이 아니라
    실제 게임에서 처럼 파일 1, 2, 3을 만들어서 선택해서 저장할 수 있도록 변경
    args를 활용하여 프로그램 시작 시 입력한 파일로 자동 불러오기 기능을 추가하는 것도 좋을 것 같음.
    아무것도 입력 안하면 새 게임
    파일 번호를 입력하면 해당 파일에 저장된 게임을 불러오는 방식으로.

    아니면 프로그램 시작하고 선택하게 만들어도 괜찮을듯.
